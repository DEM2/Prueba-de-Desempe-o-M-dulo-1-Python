## Students Registration System

This project consist of creating a simple students system for a school to register the basic student information  and storing the information in a csv file.

## Features

the project is Structured into the following way:

- **app.py**  
  Entry point of the application. It initializes and runs the system.

- **processes.py**  
  Contains the core functionalities of the system, including students registration, display students list, search a student with id , student data deletion , and updates student data.

- **validations.py**  
  Provides input validation utilities to ensure data integrity, such as numeric checks and non-empty inputs.

- **csv_manager.py**  
  Manages data persistence by handling reading from and writing to CSV files.

## Data Files

The system uses CSV files to store inventory data.

- `students.csv` → Main file used to store product data  

## How to Run the Project 

```bash 
python3 app.py
```
